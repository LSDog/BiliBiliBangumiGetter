# BiliBiliBangumiGetter
爬点儿b站番剧列表

# 使用方法
打开 `BiliBangumiGetter.py`  
修改 `md_start` 和 `md_end` 为你想要爬的开始和结束media号  
使用命令行工具运行 `BiliBangumiGetter.py`  
所有爬到的都会写进 `MediaList.txt` 里面
